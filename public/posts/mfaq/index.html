سوالات متداول الایت موشن#

آموزش ساخت اکانت در الایت موشن
مشاهده آموزش تصویری

رفع ارور 3565 در الایت موشن

  • روش اول: ۵ دقیقه صبر کنید، خودش رفع میشود
  • روش دوم: فیلترشکن روشن کنید و وارد برنامه شوید

علت کار نکردن دکمه‌های الایت موشن
این مشکل مربوط به باگ نسخه‌های جدید است:

  • صبر کنید تا رفع شود
  • از برنامه خارج شوید، کش را پاک کرده و مجدد وارد شوید
  • از نسخه‌های قدیمی‌تر استفاده کنید

⚠ هشدار مهم: قبل از حذف برنامه از پروژه‌های خود بک‌آپ بگیرید. همه پروژه‌ها را انتخاب کرده و به صورت فایل XML خروجی بگیرید و در تلگرام ذخیره کنید.

رفع ارور Update Available
دسترسی برنامه به اینترنت را مسدود کنید:

  1. تنظیمات گوشی → Programs → Alight Motion
  2. بخش Data usage → گزینه Background data را خاموش کنید

سوالات متداول Node Video#

محدودیت نصب روی دستگاه‌های 32 بیتی

  • فقط روی دستگاه‌های 64 بیتی قابل نصب است
  • اگر نصب نمی‌شود یا بعد از نصب بسته می‌شود، برنامه را پاک کنید

بررسی پشتیبانی دستگاه برای بررسی اینکه گوشی شما 64 بیتی است یا نه:

  • نصب برنامه‌های CPU-Z یا Droid Info
  • در بخش CPU بررسی کنید:
    • arm64-v8a : گوشی 64 بیتی (قابل نصب)
    • armeabi-v7a : گوشی 32 بیتی (غیرقابل نصب)
