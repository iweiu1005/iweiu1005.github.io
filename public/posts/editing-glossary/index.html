اصطلاحات پرکاربرد در ادیت

لیست پرکاربردترین اصطلاحات و کلمات در ادیت عکس و فیلم
نوامبر 22, 2025 · 4 min · تارا | آپدیــت

📂 دسته‌بندی اصطلاحات ادیت#

1) ابزارهای پایه#
  • Cut بریدن بخشی از ویدیو یا صدا
  • Trim حذف قسمت‌های اضافه از ابتدا یا انتها
  • Split تقسیم ویدئو به چند بخش
  • Crop برش بخشی از تصویر/ویدیو
  • Resize تغییر اندازه کل تصویر
  • Scale تغییر اندازه لایه
  • Rotate چرخش لایه
  • Flip برعکس‌کردن تصویر (آینه‌ای)
  • Clip یک تکه ویدیوی مجزا
  • Scene یک بخش داستانی از ویدیو
  • Frame هر تصویر جداگانه از ویدیو
  • FPS (Frames per Second) تعداد فریم در ثانیه
  • Freeze Frame نگه‌داشتن یک فریم به‌صورت ثابت
  • Reverse پخش ویدیو به عقب
  • Slow Motion حرکت آهسته ویدیو
  • Smooth Slow motion حرکت آهسته و نرم ویدیو
2) تایم‌لاین و مدیریت پروژه#
  • Timeline نوار زمانی ویدیو
  • Track لاین‌های صوتی/تصویری در timeline
  • Timeline Marker علامت‌گذاری یا نشانه‌گذاری
  • Import وارد کردن فایل
  • Export خروجی گرفتن از پروژه
  • Render آماده‌سازی پروژه برای خروجی
  • Format نوع فایل خروجی (MP4, MOV…)
  • Resolution وضوح تصویر (720p, 1080p, 4K…)
  • Template قالب آماده
3) لایه‌ها و ترکیب تصاویر#
  • Layer لایه‌ای برای قرار دادن افکت یا المان
  • Opacity میزان شفافیت
  • Overlay قرار دادن لایه‌ها روی هم
  • Picture-in-Picture (PiP) نمایش دو ویدیو هم‌زمان
  • Split Screen نمایش چند بخش تصویر کنار هم
  • Background پس‌زمینه
4) ماسک و تکنیک‌های پیشرفته#
  • Mask / Masking بریدن یا افکت‌دادن به بخشی از تصویر
  • Motion Tracking دنبال‌کردن یک سوژه
  • Stabilization حذف لرزش دوربین
  • Keyframe نقطه‌ای برای تغییر/حرکت
  • Keyframe Animation ساخت انیمیشن با تغییرات فریم‌به‌فریم
5) رنگ، نور و اصلاح تصویر#
  • Color Correction اصلاح رنگ
  • Color Grading دادن حس و استایل رنگی (سینمایی و…)
  • Brightness روشنایی
  • Contrast کنتراست (اختلاف نور)
  • Saturation شدت رنگ
  • Vibrance افزایش رنگ بدون اشباع بیش‌ازحد
  • Exposure نوردهی کلی
  • Shadows تنظیم نواحی تیره
  • Highlights تنظیم نواحی روشن
  • Tone Curve تنظیم دقیق تیرگی/روشنی
  • Hue تغییر کلی رنگ‌ها
  • Temperature گرم/سردکردن رنگ
  • Tint ته‌رنگ سبز یا ارغوانی
  • White Balance تنظیم رنگ سفید
  • RGB Channel کنترل قرمز، سبز و آبی
  • HDR ترکیب چند نوردهی برای جزئیات بیشتر
6) افکت‌ها و فیلترها#
  • Filter فیلترهای آماده
  • Motion Blur تاری حرکتی
  • Blur محوکردن بخش‌هایی از تصویر
  • Sharpness افزایش وضوح
  • Clarity قوت بافت‌ها
  • Texture شدت نمایش جزئیات ریز
  • Vignette تیره/روشن کردن لبه‌ها
  • Grain نویز شبیه عکس فیلمی
  • Noise Reduction حذف نویزها
7) حذف پس‌زمینه و ترکیب پیشرفته#
  • Green Screen پس‌زمینه سبز
  • Chroma Key حذف یک رنگ خاص
  • Masking انتخاب بخش خاص برای افکت دادن
  • Crop & Zoom برش و بزرگ‌نمایی یک قسمت
8) متن و المان‌های گرافیکی#
  • Text متن قابل ویرایش
  • Title عنوان یا تیتر
  • Watermark لوگو یا نوشته ثابت روی ویدیو
9) صدا#
  • Audio Sync هماهنگی صدا و تصویر
  • Volume حجم صدا
  • Fade In/Out (Audio) محو شدن یا ظاهر شدن تدریجی صدا
  • Noise Reduction (Audio) حذف نویز صدا
  • Equalizer (EQ) تنظیم فرکانس‌های صدا
  • Beat بیت
  • Beat Marking بیت‌گذاری، علامت‌گذاری ضرب‌های مهم موسیقی روی تایم‌لاین تا برش‌ها، ترنزیشن‌ها و افکت‌ها دقیقاً روی ریتم موزیک قرار بگیرند
  • Audio Gain افزایش قدرت/ولوم صدا
10) ترنزیشن‌ها#
  • Transition افکت بین دو کلیپ
  • Fade محو و ظاهر شدن تدریجی تصویر
  • Zoom In/Out Transition ترنزیشن بزرگ‌نمایی
  • Slide / Wipe اسلایدی یا پاک‌شونده 
  • Luma Fade ترنزیشن با روشنایی 
11) ابزارهای انتخاب، جابجایی و مدیریت لایه‌ها #
  • Transform — جابجایی، تغییر اندازه، چرخش و کنترل موقعیت لایه
  • Lasso Tool — ابزار انتخاب آزاد برای تعیین ناحیه دلخواه
  • Brush — قلمو برای نقاشی، اصلاح یا افکت دادن
  • Bucket / Fill Tool — پرکردن نواحی با یک رنگ
  • Eyedropper — قطره‌چکان برای برداشتن رنگ از تصویر
  • Canvas — بوم اصلی پروژه (محیط کلی کار)
12) ابزارهای انتخاب و طراحی#
  • Transform جابجایی، چرخش و تغییر اندازه لایه
  • Lasso Tool انتخاب آزاد مناطق تصویر
  • Brush قلمو برای نقاشی، ماسک‌کردن، روتوش
  • Bucket / Fill Tool پرکردن ناحیه‌ای با رنگ
  • Eyedropper برداشتن رنگ از هر نقطه
  • Canvas بوم/صفحه اصلی پروژه
  • Eraser پاک‌کن 
  • Smudge کشیدن و مخلوط‌کردن پیکسل‌ها 
  • Clone Stamp / Clone Tool کپی‌کردن بافت از بخش دیگر
  • Liquify تغییر فرم و شکل المان‌ها
اصطلاحات دیگه#
  • Proxy نسخه سبک‌تر ویدیو برای ادیت روان‌تر
  • Aspect Ratio نسبت تصویر (مثلاً 16:9 / 9:16)
  • Color LUT فایل تنظیمات رنگ آماده (برای گریده)
  • Motion Graphics عناصر گرافیکی متحرک
  • Blend Mode نحوه ترکیب لایه‌ها (Screen, Multiply…)

این لیست رو تارا زحمتش رو کشیده و توش به اون چیزایی اشاره کردیم که توی ادیت زیاد باهاشون سروکار داریم. امیدواریم به درد کسایی بخوره که تازه دارن ادیت رو شروع می‌کنن و کمک کنه راحت‌تر با ابزارها و اصطلاحات مختلف آشنا بشن.

 برای دیدن آموزش و ترفند های بیشتر به چنلمـــــون سر بزن !
